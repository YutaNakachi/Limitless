# Limitless

![Platform](https://img.shields.io/badge/Platform-PC%20%2F%20WebGL-orange)

**「様々な能力を持つ呪術球を蹴り分け、無限のステージを攻略する超常スタイリッシュアクション！」**

『Limitless』は、王道アクションの心地よい手触りと、ガチャの運要素を融合させた、2Dスタイリッシュ・タクティカルシューティングアクションゲームです。

---

## 🔮 ゲームコンセプト
- **馴染み深くキレのある操作性**
  - 移動、ジャンプ、ダッシュなどの基本挙動は、直感的かつ「きびきびとしたスピード感」を徹底的に再現。
- **呪術球（ボール）を蹴り出す独自の攻撃**
  - 自身の周囲を公転する球をタイミングよく蹴り飛ばし、着弾時に戦況を大きく変える特殊能力を発動。
- **戦術に即興性を生む「ガチャ要素」**
  - リロードごとにランダム装填される手札（強力な球ほど低確率）に応じ、その場で最適な立ち回りを瞬時に組み立てる即興の戦術性。

## 🛠️ 特徴的なゲームシステム

### 1. 呪術球・オービットシステム（Orbit System）
プレイヤーの周囲を公転し続ける6つの呪術球（固定スロット管理）。
- **属性（一例）**: 「無色（直線弾）」「赤色（爆発）」「青色（吸引）」
- **目押しメカニクス**: プレイヤーは**自身の正面（キックポイント）に回ってきた球だけ**を蹴り出すことが可能。回るタイミングを冷徹に見極め、狙いの色を撃ち抜く「運」と「技術」の融合システム。

### 2. 視覚的演出（VFX）
- **未来予測**: 次に放つ球から射線を捉える `LineRenderer`。
- **航跡演出**: 蹴り出された球は、ロケットのようなモコモコとした白い煙（Particle）と、鮮やかな光の残像（Trail）を引き連れて滑らかに飛行。
- **身体的爽快感**: 回避時や撃破時の絶妙な「ヒットストップ（一瞬のストップ演出）」による極上の手触り。

## 🎮 搭載ゲームモード

| モード名 | 形式 | 概要 |
| :--- | :--- | :--- |
| **リミットレス・サバイバル** | 1人 | 無限に自動生成され、リニアに難易度が上昇するステージの限界突破（最高クリア数）に挑むストイックなモード。 |
| **超常共闘モード** | 協力（2人〜） | リロードの隙をカバーし合い、「青で集めて赤で仕留める」といったマルチ特有のド派手な連携コンボを楽しむモード。 |
| **オービット・バトル** | 対戦（2人） | 敵なしのガチンコ2D対戦。相手の周囲の球（手札）を読み合い、死角から目押しの一撃を叩き込む極限の心理戦。 |

## ⌨️ 基本操作仕様
- **移動**: `A` / `D` キー（左右移動）
- **しゃがみ**: `S` キー
- **ジャンプ**: `W` キー（空中ジャンプを含め、最大2段まで可能）
- **ダッシュ**: `A` または `D` キーを**すばやく2回入力**
- **キック**: `マウス左クリック` （正面のキックポイントに来た球を射出）
- **オンタイム・リロード**: `R` キー（空きスロットへランダム装填）

## 🛠️ 開発環境 / プロジェクト構成
- **Engine**: Unity 2022.3 LTS 移行（2D URP推奨 / Post-Processing (Bloom) 使用）
- **Language**: C# (オブジェクト指向思想に基づく設計。`BallAbility` 基底クラスによる拡張構造を想定)
